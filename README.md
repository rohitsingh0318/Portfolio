## 📌 About the Project

This portfolio is designed to highlight my work as a full-stack developer. It includes real-world projects, clean UI design, and smooth navigation to provide a great user experience.

---

## ✨ Features

* 🎨 Modern and clean UI design
* 📱 Fully responsive (mobile, tablet, desktop)
* ⚡ Fast performance and optimized loading
* 🧩 Dynamic project section using JSON data
* 🔗 Live project preview and GitHub links
* 🖼️ Image-based project showcase
* 🎯 Smooth navigation and user-friendly layout

---

## 🛠️ Tech Stack

* **Frontend:** React.js, JavaScript
* **Styling:** Tailwind CSS / CSS3
* **Icons:** React Icons
* **Deployment:** Vercel
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/rohitsingh0318/Portfolio.git
```

Go to project folder:

```
cd Portfolio
```

Install dependencies:

```
npm install
```

Run locally:

```
npm run dev
```
